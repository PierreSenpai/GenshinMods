Change emojis...
 1. Browse the PNG folder to see them all
 2. Edit "emojis.ini" to change the numbers
    (Keep them as .dds - no need to use .png!)

Better way to change emojis and keep the changes...
 1. Make a copy of the emojis file and rename it to @swap
 2. Delete everything in it except for "namespace = paimon"
    and the characters you plan to change the emojis of
 3. Carry the @swap file with you between versions
